マリトッツォ大好きブログ

このサイトについて

このサイトは、２０２３年東京都立大学のネットワーク演習・実習Ⅰの課題として制作したサイトです。
タイポグラフィを題材に、ホラーサイトを作りました。 ブログ型のホラーサイトであり、物語が進むにつれてサイト自体も変容していきます。 タイポグラフィを巧みに使ったホラー表現にも挑戦し、文字の配置やアニメーションにも工夫を凝らしています。 ぜひ注目してみてください。

内容について

2023年6月現在、街でマリトッツォがほぼ売られていないという情報を耳にしました。
それを踏まえ、マリトッツォが大好きな女子高生が、だんだんマリトッツォが販売されなくなっていき、 食べたい気持ちが恨みに変わっていく・・・というストーリーを考えました。 本当に怖いサイトにはせず、マリトッツォという少しコミカルな要素を入れたつもりです。
ブログの内容は、ChatGPTにそれぞれの日のタイトルを渡し、本文を考えてもらいました。 実際の情勢に即して、ブログの内容が書かれているので、マリトッツォのブームについても知っていただくことができます。
